Hello World！

发表于 2018-03-10 | 分类于 技术

　　经过这几天的捯饬，博客终于算是成形了。这个hexo + github博客项目入门其实并不是很难，因为没有后台，而且有很多好看的模板，只需要一些简单的配置即可。但是细节知识以及原理其实还不懂，不过好在有了这个属于自己博客，以后就可以把自己的一些学习笔记、心得体会放在上面了。万事开头难，好在这个头已经开了，接下来要趁年轻，更加努力地去实现自己的梦想啊！

　　下面是搭建这个博客的一个简单手册，放在这里，以防止下次再搭建又要各种上网找资料。

下载并安装git for windows

1.地址：https://gitforwindows.org/
2.选择安装路径后，next直至finish

下载并安装Node.js

1.地址：https://nodejs.org/en/
2.LTS为长期支持版，Current为当前最新版
3.选择安装路径后，next直至finish

安装hexo

1.新建文件夹，存放博客所有的文件
2.安装hexo：
    在该文件夹上右击选择Git Bash Here，输入命令：
    npm i -g hexo
    耐心等待即可
3.输入初始化命令：
    hexo init

生成SSH并添加到github

1.在github上创建repository，名称必须为yourname.github.io，yourname指github名称
2.在git bash中配置github账户信息，输入命令：
    git config --global user.name "RadiationJIE"
    git config --global user.email "826922042@qq.com"
3.创建SSH
    在git bash 中输入命令：
    ssh-keygen -t rsa -C "826922042@qq.com"
    回车直至结束，找到id_rsa.pub路径，复制id_rsa.pub中的内容
4.在github的Settings-SSH and GPG keys中New SSH key，起个title名，key中填入复制内容后
Add SSH key
5.验证SSH key是否添加成功，输入命令：
    ssh -T git@github.com
    输入yes

部署项目

1.修改blog文件夹下_config.yml文件的一些配置(文件末尾）：
deploy:
  type: git
  repo: https://github.com/YourgithubName/YourgithubName.github.io.git
  branch: master
2.回到git bash，进入blog目录，分别执行以下命令：
    hexo clean
    hexo generate
    hexo server
    执行后可在浏览器中输入http://localhost:4000进行验证；在命令行中ctrl+C退出（这时就无法
    访问http://localhost:4000了）
3.hexo 3.0把服务器独立成个别模块，需单独安装，输入命令：
    npm i hexo-server
    耐心等待即可

上传到github

1.为了让你写好的文章部署到github服务器上并让别人浏览到，先输入命令安装：
    npm install hexo-deployer-git --save
2.每次都按以下步骤部署，输入命令：
    hexo clean
    hexo generate
    hexo deploy
    deploy过程中需要输入你的username和password
在浏览器中输入http://yourgithubname.github.io即可看到你的个人博客。

主题配置及修改

将主题从github中下载下来放入hexo的themes中，在_config.yml中修改theme为下载的主题文件夹名
称。
重新部署：
    hexo clean
    hexo generate
    hexo deploy
hexo s –debug 可进入调试模式，浏览器输入http://localhost:4000/
